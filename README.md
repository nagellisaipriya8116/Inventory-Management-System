# 📦 Inventory Management System

## 📖 Project Description

The Inventory Management System is a Python-based application developed to simplify inventory management for businesses. It helps users manage products, suppliers, purchases, sales, and stock information efficiently using an SQLite database.

---

## ✨ Features

* User Login Authentication
* Product Management
* Supplier Management
* Purchase Management
* Sales Management
* Low Stock Alert
* Reports Generation

---

## 🛠 Technologies Used

* Python
* SQLite
* VS Code

---

## 📂 Project Structure

* `database.py` – Creates database and tables
* `login.py` – User authentication
* `main.py` – Main menu
* `product.py` – Product management
* `supplier.py` – Supplier management
* `purchase.py` – Purchase module
* `sales.py` – Sales module
* `reports.py` – Reports generation
* `low_stock.py` – Low stock alerts

---

## 🚀 How to Run

1. Install Python.
2. Open the project in VS Code.
3. Run `database.py` (only the first time).
4. Run `login.py`.
5. Enter your username and password.
6. Use the Inventory Management System.
